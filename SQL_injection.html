SQL_injection

Es un tipo de ataque cibernetico que se aprovecha de vulnerabilidades en las aplicacion que utilizan base de datos para inyectar codigo SQL malicioso

Puedn tener graves concecuencias como:

  • Robo de datos sencibles: el atacante puede acceder a datos confidenciales, como numeros de tarjeta de credito, contraseñas o informacion personal.
  • Modificacion de datos: El atacante puede obtener acceso a cuentas de administrador o super usuario, lo que le permite realizar acciones no autorizadas .
  • Denegacion de servicios el atacante pude causar que la aplicacion o la base de datos se vuelvan inaccesibles.

¿Comó pevenir los ataques?

  1. Validar y sanizar los datos de entrada
  2. Utilizar consultas SQL preparadas y parametrizadas
  3. Limitar los privilegios de la cuenta de base de dados
  4. Mantener la base de datos y la aplicacion actualizada y parchada
