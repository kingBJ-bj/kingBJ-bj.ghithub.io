Cross-Site

Es un tipo de ataque cibernetico que se aprovecha de una vulnerabilidad de una aplicacion web para inyectar un codigo malicioso en la pagina web de un usuario

:

  • envian una gran cantidad d trafico de red al sistema para sobrecargarlo
  • utilizan servidores legitimos como DNS o correo electronico para amplificar el trafico de red.
  • ataques de aplicacion, se dirigen aplicaciones especificas como servidores web o base de datos para sobrecargarlas.

¿Comó protergerse contra los ataques DDoS?

  1. Utilizar firewalls y sistema de detencion de intrusos
  2. implementar tecnicas de mitigacion de DDoS como la limpieza del trafico y la redireccion de trafico
  3. mantener los sistemas y las aplicaciones actualizados y parchados
  4. ralizar copias de seguridad regulares de los datos importantes
