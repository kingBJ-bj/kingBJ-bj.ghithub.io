¿Qué es el phishing?

El phishing es un tipo de ataque en el que un ciberdelincuente se hace pasar por una entidad confiable, como un banco o red social, para obtener información confidencial como contraseñas o datos bancarios.

Métodos comunes

  • Correos electrónicos falsificados.
  • Mensajes de texto sospechosos.
  • Sitios web que imitan páginas legítimas.

¿Cómo protegerte del phishing?

1. Verifica las URL

Antes de hacer clic en un enlace, revisa que la dirección web sea legítima.

2. Activa el doble factor de autenticación

Este paso agrega una capa extra de seguridad a tus cuentas.

3. Nunca compartas datos confidenciales

Los bancos o instituciones legítimas nunca te pedirán contraseñas o códigos por correo.
